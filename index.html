OpenWiFiMap history: André's motivation

  • Before 2012: freifunk in Berlin had 3 (or more) maps:
    • ffmap
    • heartbeat
    • yaffmap
  • In 2012: all maps and node databases were gone due to a server crash!
  • If a service can be shut down completely by simply disabling one server, perhaps it wasn't good anyway! ☺

OpenWiFiMap history: Goals

Database

  • open for all communities worldwide
  • decentralized
  • replicable
  • flexible data format
  • simple HTTP REST API
  • scalable

Map visualization

  • show nodes+links
  • desktop- and touch-devices
  • global view possible
